This project analyzes bank loan data using Excel, Power Query, and Power Pivot, Tableau and power BI to transform raw data into interactive dashboards and meaningful insights. 
The analysis focuses on loan approvals, customer behavior, branch performance, and collection status to support better business decisions
The aim of this project is to analyze loan data and extract key insights that help in:
Evaluating loan performance 
Monitoring branch efficiency 
Understanding customer behavior 
Supporting data-driven decision-making through KPIs and dashboards
